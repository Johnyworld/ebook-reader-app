# 다국어(i18n) 지원 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 앱 UI 텍스트를 Android 표준 리소스 시스템으로 전환하여 5개 언어(영어, 한국어, 일본어, 중국어, 스페인어)를 지원한다.

**Architecture:** 하드코딩된 ~150개의 한글 문자열을 strings.xml로 추출하고, enum label을 `@StringRes`로 전환한다. 메인 화면 헤더에 언어 선택 드롭다운을 추가하고, `AppCompatDelegate.setApplicationLocales()`로 언어를 전환한다.

**Tech Stack:** Android Resource System (strings.xml), Jetpack Compose (`stringResource()`), AppCompatDelegate, Per-App Language API (Android 13+)
