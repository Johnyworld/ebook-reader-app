## 한국어

### 짧은 설명 (80자 이내)
EPUB, PDF, MOBI, TXT를 지원하는 가볍고 빠른 오프라인 전자책 리더

### 전체 설명
Ebook Reader는 기기에 저장된 전자책을 편하게 읽기 위한 오프라인 리더 앱입니다.

📖 지원 형식
EPUB, PDF, MOBI, TXT 파일을 지원합니다. 기기의 저장소를 자동으로 스캔하여 전자책을 찾아줍니다.

✏️ 독서 도구
• 하이라이트: 중요한 문장에 하이라이트 표시
• 메모: 원하는 곳에 메모 작성
• 북마크: 원하는 페이지를 북마크로 저장
• 본문 검색: 책 안에서 텍스트 검색
• 목차: 챕터별 빠른 이동

⚙️ 맞춤 설정
• 글꼴 크기, 줄 간격, 문단 간격 조절
• 상하좌우 여백 조절
• 텍스트 정렬 방식 변경 (양쪽, 왼쪽, 오른쪽, 가운데)
• 페이지 넘김 방향 설정 (좌우/상하)
• 외부 폰트 가져오기
• 가로모드 두 쪽 보기
• 하단 정보 표시 (페이지, 진행률, 시계 등)

📚 서재 관리
• 즐겨찾기
• 숨기기
• 제목순, 작가순, 최신순 등 다양한 정렬
• 제목/저자 검색

🌍 다국어 지원
한국어, English, 日本語, 中文, Español

## English

### Short description (under 80 chars)
A lightweight offline e-book reader for EPUB, PDF, MOBI, and TXT files

### Full description
Ebook Reader is an offline reader app for comfortably reading e-books stored on your device.

📖 Supported Formats
Supports EPUB, PDF, MOBI, and TXT files. Automatically scans your device storage to find e-books.

✏️ Reading Tools
• Highlights: Mark important passages
• Memos: Write notes anywhere in the book
• Bookmarks: Save pages for quick access
• In-book search: Search text within the book
• Table of contents: Jump between chapters

⚙️ Customization
• Adjust font size, line height, and paragraph spacing
• Adjust vertical and horizontal margins
• Change text alignment (justify, left, right, center)
• Configure page flip direction (left-right / top-bottom)
• Import custom fonts
• Dual-page view in landscape mode
• Bottom bar info display (page, progress, clock, etc.)

📚 Library Management
• Favorites
• Hide books
• Sort by title, author, date added, last read, and more
• Search by title or author

🌍 Multi-language Support
한국어, English, 日本語, 中文, Español

🔒 Completely Offline
Works without an internet connection. No personal data is collected.
